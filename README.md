# DIET GASLIGHTING 😈✨

파스텔 속에서 당신의 영혼을 털어버릴 달콤한 악마, DIET GASLIGHTING에 오신 것을 환영합니다.

## 프로젝트 소개
이 앱은 사용자가 올린 식단 사진을 AI가 분석하여 영양 정보를 제공하는 앱입니다. 하지만 일반적인 앱과는 다릅니다. 아주 독설적이고 냉철한 '팩트 폭격'을 통해 다이어트 의지를 (강제로) 불태워 드립니다.

## 주요 기능
- **AI 식단 분석**: 최대 3장의 사진을 동시에 분석하여 칼로리 및 영양소 파악
- **팩트 폭격 (Fact-Attack)**: 디자인에 속아 정신 못 차리는 사용자에게 날리는 매콤한 독설
- **맞춤형 운동 추천**: 섭취한 칼로리를 속죄하기 위한 다양한 운동법(버핏, 등산, HIIT 등) 제안
- **가스라이팅 로그**: 날짜별로 기록된 당신의 잔혹한 식사 역사를 달력으로 확인

## 기술 스택
- **Frontend**: React, Vite, TypeScript
- **Styling**: Vanilla CSS, Framer Motion (Animations)
- **AI**: OpenAI GPT-4o API
- **Icons**: Lucide React

## 배포 방법 (Vercel)
1. GitHub 저장소에 코드를 업로드합니다.
2. Vercel에서 이 프로젝트를 임포트합니다.
3. 환경 변수에 `VITE_OPENAI_API_KEY`를 추가합니다.
